# Sales Data Mini-EDA

## Project Description
Mini Exploratory Data Analysis (EDA) project on monthly sales data for multiple products.  
Purpose: Analyze sales trends, visualize data, and extract key insights using *Python, **pandas, **matplotlib, and **seaborn*.

## 
Dataset
The dataset is stored in SALES.csv and contains monthly sales information for different products.  

It has three main columns:  

- *Month* – Indicates the month of the sales, e.g., Jan, Feb, Mar, Apr.  
- *Product* – Name of the product, such as Mobile, Laptop, or Tablet.  
- *Sales* – Numeric value representing the sales amount for that product in the given month.  

*Example entries from the dataset:*  

Jan, Mobile, 5000  
Jan, Laptop, 7000  
Jan, Tablet, 3000  
Feb, Mobile, 6000  
Feb, Laptop, 8000  
Feb, Tablet, 3500
